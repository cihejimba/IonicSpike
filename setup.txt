Install:
- Ionic 	        sudo npm install -g cordova ionic
- Xcode 	        app store
- Gulp 		        sudo npm install -g gulp
- iOS sim 	        npm install -g ios-sim

Init project:
- Initialize app    ionic start IonicSpike sidemenu
- Add platform      ionic platform add ios
- Build platform    ionic build ios
- Serve in browser  ionic serve
- Emulate plaform   ionic emulate ios --target="iPad (Retina)" --livereload --consolelogs --serverlogs
